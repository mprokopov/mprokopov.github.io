16.01.2011

Apple Magic Trackpad, мой опыт использования

06.01.2011
Повелся на статью Macpages и перед новым годом приобрел apple trackpad, об опыте использования которого и спешу рассказать. Во-первых использование magic trackpad это уже определенный разрыв шаблона после 20ти летнего использования манипулятора типа «мышь». Во-вторых преимущество трекпада, которое было широко разрекламировано — мультижесты, обернулось по началу большим минусом. iTerm2 для Mac

05.01.2011
По наводке г-на Бобука из твиттера была выужена ссылка на iTerm2 в alpha (хотя по-моему можно уже даже из беты переводить в релиз кандидат).
