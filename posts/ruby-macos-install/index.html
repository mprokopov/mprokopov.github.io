Install Ruby 3.1.4 on macOS

29.08.2023
Reading time: 1 min.

In case ruby can’t find OpenSSL 3 headers, use this way.

brew install ruby-install
ruby-install ruby 3.1.4 -- --with-openssl-dir=$(brew --prefix openssl)
