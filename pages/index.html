I beleive being creative is an essential part of being a human.

Before I got my first computer Intel 386 DX 40 in 1994, I used to write small programs using MK61 programmable calculator, then a bit of soviet Corvette computer programming at school. Since that time I didn’t stop my journey with software engineering, but nowadays with more focus towards DevOps and SRE practices.

My first book about software engineering was C++ by Bjarne Stroustrup. I didn’t have any computer when I read it!

First public website experience started with first web design attempts. I remember very well that excitement came from created flying and bouncing text animated by Javascript in Netscape Navigator!

One of the remarkable side-projects was a website about unwanted pregnancy prevention methods. Recently I did some efforts to reviwe it here condom.org.ua.

About

08.12.2020

Hi, my name is Maksym Prokopov. These days I mostly involved in DevOps, Lean IT and IT services.

A long time ago I started by web developer career from PHP 3.0.3 and have used lots of technlologies and programming languages. Surprisingly, but I started using PHP for my job today again after almost 18 years. What a destiny!

Since 2006 I started using Ruby on Rails a lot. 2015 got addicted to Clojure and ClojureScript and did some production-ready things.

In 2007 with partner Alexander we launched IT-Premium company, that delivers best IT-support services for organizations.

In 2016 with another partner Alexey we launched internet-shop Super-Truper.com.ua. 2018 I sold my 50% share to Alexey.

Delivered several talks on IT conferences, examples: «Стальной бубен» Lviv, Ruby Meetup Kyiv, ITSM Days Kyiv, «V Хабравстреча» Kyiv, «ИТ для бизнеса» в Клубе Деловых Людей, Kyiv Clojure Workshop.

In 2020 I managed to deliver DevOps course.

Since 2020 I moved from Kyiv to Freiburg in Greisgau, Germany.

The most important thing for a human being I believe is to be happy and share happiness with environment, and to increase kindness.

I appreciate your comments and reviews if anything I wrote about turned to be beneficial for you.
